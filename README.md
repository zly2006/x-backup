## Usage

### Commands
```
/xb create [comment]
```
Creates a backup with an optional comment.

```
/xb list
```
Lists all backups.

```
/xb restore <id>
```
Restores a backup with the specified ID.

```
/xb delete <id>
```
Deletes a backup with the specified ID.

```
/xb info <id>
```
Displays information about a backup with the specified ID.
